**Summary:**

Praxis-Cloak is a Python-based repository that provides an on-device solution for protecting Personal Identifiable Information (PII) when interacting with Cloud Large Language Models (LLMs). The project uses the Ollama framework to detect and replace sensitive entities in user prompts, sending scrubbed prompts to cloud models. The response from the cloud model is then mapped back to the original PII entities, ensuring that users receive accurate answers while protecting their sensitive data.

**Key innovations:**

The key innovation of Praxis-Cloak lies in its round-trip substitution approach, which intercepts user prompts before they leave the device, detects and replaces PII with realistic stand-ins, sends scrubbed prompts to cloud models, and maps the answer back to original entities. This approach significantly reduces PII exposure while maintaining answer quality.

* Frameworks and Dependencies: Ollama, HuggingFace (for downloading model weights), Pyproject

**Potential impact:**

Praxis-Cloak has the potential to make routine LLM use substantially less leaky while maintaining answer quality, reducing the risk of PII exposure in cloud-based AI interactions. However, it's essential to note that this is not a full-fledged privacy guarantee, and determined adversaries with enough queries could still reconstruct context. The project aims to provide a practical solution for disclosure control, making it an interesting contribution to the AI/ML ecosystem.

Here is a concise summary of the TokenTamer GitHub repository:

**Project Overview**

TokenTamer is an intelligent drop-in middleware proxy designed to compress bloated code context in real-time, cutting Large Language Model (LLM) API costs by 50-80% without compromising what the model needs to know.

**Key Features**

1. **Drop-in proxy**: No changes needed to coding agents; simply update the API base URL.
2. **Smart active file detection**: Automatically identifies files being worked on and leaves them intact.
3. **AST-based compression**: Strips function bodies while preserving signatures, imports, and class structures.
4. **Real-time cost tracking**: Displays tokens saved and money saved in a beautiful terminal dashboard.

### Summary

**1. What it does**
Local Recall is an early prototype of a fully local, privacy-first alternative to Microsoft Recall. It captures screen snapshots, extracts text using WinRT OCR, generates vector embeddings, and allows users to conversate with their screen history using a local Large Language Model (LLM).

**2. Key innovations**
What makes Local Recall notable is its focus on complete locality, running entirely on the user's own machine without any cloud communication. This approach provides an attractive alternative for those concerned about data privacy and security. Additionally, the project's innovative use of the WinRT OCR Engine and local vector embeddings enables efficient text extraction and semantic search.

### Summary

Here is a concise summary of the GitHub repository:

**Project Overview**

The 0xquqi/crypto-kol-quant repository is an open-source project that distills 99 crypto KOL trading experiences into backtestable quant factors using LLM (Large Language Model) technology. The goal is to create a set of rules-based trading strategies based on the trading insights and habits of top crypto traders.

**Key Features**

* Extracts 470 "transactional intuition" points from 39,843 tweets and 17,657 K-line screenshots.
* Transforms these points into Python evaluation functions for 87 quant factors.
* Provides historical price data for 832 days.
* Offers real-time consensus output through the `/consensus` command.

**Backtesting Results**

The project includes backtesting results on BTC and ETH using a 7-day period, showing an accuracy of 57% in following the trading signals. Another test using a 5-minute entry strategy resulted in an accuracy of 59%.

**Usage**

The project provides two usage methods:

1. **Claude Code skill**: A set of commands to use with the `/consensus` command for real-time consensus output.
2. **Manual analysis**: Users can analyze and customize the quant factors using Python.

Overall, this project aims to provide a practical tool for traders to leverage top crypto trader insights into automated trading strategies.

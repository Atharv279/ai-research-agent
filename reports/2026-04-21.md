The Wraith project presents a novel architecture called Native Pure Quantized Network (NPQN), which combines three distinct properties for the first time in Large Language Models (LLMs): native quantization, pure arithmetic, and 9-level dualwire quantization. The model is trained from scratch with an integer pipeline and achieves state-of-the-art results.

**2. Key innovations:**
The Wraith project introduces a new architecture that integrates three innovative aspects:
* Native quantization, which trains the model directly from initial random weights without post-hoc quantization.
* Pure arithmetic, which operates entirely in fixed-point arithmetic with no floating-point numbers.
* 9-level dualwire quantization, which achieves an optimal of Shannon for two-channel ternary weights.

These innovations make Wraith a pioneering work in LLM research.

**3. Tech stack:**
The primary language used is TeX, and the tech stack consists solely of TeX (100% detected). No other frameworks or dependencies are mentioned.

**4. Potential impact:**
The introduction of NPQN architecture has the potential to significantly impact the AI/ML ecosystem:
* Wraith's native quantization and pure arithmetic could lead to improved efficiency, reduced memory requirements, and faster inference times.
* The 9-level dualwire quantization could provide a new standard for ternary weights, potentially leading to better performance and scalability in various applications.

Overall, the Wraith project showcases a groundbreaking approach that challenges existing LLM architectures and has the potential to drive innovation in AI research.

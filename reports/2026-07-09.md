**README excerpt:** # AgentRule-Skill

AgentRule-Skill 是一个面向中国国内开发者的 Agent 智能体编写规则包。它用 `spec.md → plan.md → tasks.md → checklist.md` 四个评审包，把“先想清楚 Agent 怎么行动”放在“写提示词、接工具、跑评测”之前，帮助团队减少越权调用、幻觉编造、输出漂移和上线后返工。

## 项目定位

这个仓库不是通用软件开发模板，也不是某个项目的私有规则。它专注于一件事：**指导 AI 编写、改造和验收 Agent 智能体**。

适合的 Agent 包括客服分流、企业知识库问答、费用报销初审、合同条款审查、运营数据分析、终端研发助手、飞书/钉钉/企微工作流助手、小程序客服助手等。

## 解决的问题

国内团队做 Agent 时，常见问题不是“不会写提示词”，而是这些内容没有提前定清楚：

* 是否允许读文件、写文件、调接口、发消息、下单、审批或执行命令。

**README excerpt:** # AgentVideo Pipeline

AgentVideo Pipeline 是一个 Windows 优先的视频生产流水线。它面向两类使用者：普通用户可以用网页拖入素材、一步步生成视频；开发者和 Agent 可以通过 API 编排项目创建、素材导入、时间线生成、字幕、渲染和批量导出。

## 能做什么

- 拖入多个视频、音频或图片素材，只处理你选中的文件。
- 根据脚本文案生成结构化时间线和字幕。
- 导出竖屏、横屏、方形等多平台视频版本。
- 删除口癖、调整字幕、重排片段、检测静音片段。
- 保存素材清单、时间线、字幕稿、版本快照和操作记录。
- 提供 FastAPI 接口，方便 Codex、OpenClaw 或业务系统自动调用。
